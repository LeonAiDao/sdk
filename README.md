## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/leon-ai.git
cd leon-ai
```

### 2. Install Dependencies

**Node.js:**

```bash
npm install
```

**Python:**

```bash
pip install -r requirements.txt
```

---

### 3. Setup Environment Variables

Copy the example file:

```bash
cp .env.example .env
```

Edit `.env` with your configuration.

---

### 4. Run the Project

**Node.js:**

```bash
npm run dev
```

**Python:**

```bash
python main.py
```
